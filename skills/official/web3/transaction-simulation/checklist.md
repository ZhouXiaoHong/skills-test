# Transaction Simulation Checklist

Use this checklist when reviewing a simulation/preview feature.

## Must-have

- [ ] Clear token deltas (in/out)
- [ ] Approval detection (including unlimited approvals)
- [ ] Contract address display + block explorer link
- [ ] Safe defaults when decoding fails

## Nice-to-have

| Item | Why |
| --- | --- |
| Address reputation | Helps spot known scam contracts |
| Human readable actions | Improves UX for non-technical users |
| Gas + fee breakdown | Helps users estimate total cost |
